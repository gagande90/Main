Python programming language does offer some specific keywords which can be utilized for certain specific purposes. For example - break can be used to get out of a loop. It's same like a window of a car and can be opened anytime to get out of car. Let's see how exactly does this works out in Python Programming language: -
for num in range(2, 10):
     if num % 2 == 0:
          print("Found an even number", num)
          break
     print("Found a number", num)

#Output of Program
Found an even number 2 In order to understand this program we need to see how's this works exactly. Let's see how it works, range(2, 10) means numbers from 2 to 9. In the first iteration of range(2, 10) num is 2. In the next line we calculate num % 2 which evaluates to zero, thus print("Found an even number", num) statement is executed which means output would be `Found an even number 2`. The loop does not move forward owing to the reason that there is a break statement which stops loop from going on to next iteration of loop in which num would becomes 3. That's how break statement in Python works on the other hand for the same code if we have continue.Then loop continues to iterate until the end of range function is reached.
