Python is an interpreted, object-oriented, high-level programming language with dynamic semantics. Its high-level built in data structures, combined with dynamic typing and dynamic binding, make it very attractive for Rapid Application Development, as well as for use as a scripting or glue language to connect existing components together. In the higher level view of the langauge variables are stored into the memory blocks. As the python is at higher level of abstraction then syntax which is required for variable declaration is quite simple and robust. But ultimately every declaration got converted into low level C or C++ code which involves using pointers and memory addresses for storing and then keeping track of variables.
